+***********************************+
 OpenBougie 目录结构
+***********************************+
OpenBougie
→ app-----------------------APP应用目录
→ static--------------------静态数据目录
→ public--------------------公共模型目录
→ install-------------------安装程序目录
→ plugins----------------------插件目录
→ system--------------------系统核心底层
→ themes------------------------皮肤目录
→ language---------------------语言包目录
→ cache-------------------系统缓存目录[*]
→ tmp---------------------模板缓存目录[*]
→ uploads---------文件、图片上传存储目录[*]
→ favicon.ico---------------网站icon图标
→ index.php--------------------单入口文件
→ robots.txt-----------------搜索限制文件
